# Intro

将语音文件转成文字。

# 安装依赖

## 系统依赖

```bash
sudo cat system_requirements.txt | xargs sudo apt-get install -y
```

## python依赖

```bash
python -m venv .v2t
source .v2t/bin/activate
python -m pip install -r requirements.txt
```
